Purplship comes with a set of product that can be used to integrate shipping
capabilities to your apps. Developers can use the APIs to create a tailored
shipping experience for customers.

Purplship exposes:

- A Multi-carrier Shipping OpenAPI: Get started with the unified [Purplship REST API](/docs/reference/openapi)
- A Shipping App Graph API: Access your shipping data with the [Purplship GraphQL API](/docs/reference/graphql)
